# Files and Directory information
## Directory Information
<b>Parent Folder</b>
```
assets = resources
etc = other file
src = main file
include = header file
test = testing unit
tools = tools configuration

```

<b>Sub Folder</b>
```
core = core for header file
lib = lib for header file
component = component for header file
```
